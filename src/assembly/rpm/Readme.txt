Pre-requisites
***************
1)python modules requests must be previously installed
Pyhton modules can be installed:
	pip requests install
	easy_install requests install

2)The environmental variables OS_TENANT_NAME_VALUE, OS_USERNAME_VALUE, OS_PASSWORD_VALUE should be created in the jenkins machine with adequeate values to obtain token from KEYSTONE_URL
3)The sdc rpm location should be provided as parameter
